-   夏洛分享如何在三角关系中胜出的策略和态度

    2.  挖墙角涉及复杂道德负担，是多方博弈的过程

-   通过适当升级关系，但不主动提感情，保持朋友关系舒适区
    1.  在合适氛围下升级关系，同时做免责声明
    2.  等待窗口出现，抓住机会升温关系
    3.  勇敢表态，让女生选择，接受结果，坦然面对幸福与选择
-   提出挖墙角方法论，尊重女生选择，接受结果

    2.  尊重女生选择，为自己幸福考虑
    3.  希望女生过得更好，理解挣扎和痛苦，明天再见。
