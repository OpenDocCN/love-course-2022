# 【总结】安小妖《好男人，你也要学会撩妹》：231.你要的高情商，就在生活的这几个细节中

-   安小妖《好男人，你也要学会撩妹》中提到维持长期关系的细节，包括慎重交流、避免吵架、情商处理和尊重对方生活。
    1.  在关系初期慎重交流，避免过快暴露自己，保持吸引力。
    2.  吵架是一种病毒，应避免言语伤害，停止争吵，用幽默转移话题。
    3.  高情商处理情绪，找到根源满足需求，不过分干涉对方生活，接受对方不完美。
-   人们在生活中常常面对的问题是惰性，即明知道却无法做到的情况。保持尊重和礼貌可能最初是为了讨好，但当没有需求时会打折扣。意识到无条件尊重他人是真正理解生活的关键。
    1.  生活中常常要对抗的是自己的惰性，而非愚钝。
    2.  最初保持尊重可能是为了讨好，但当没有需求时会减少。
    3.  意识到并实践无条件尊重他人是真正理解生活的标志。
