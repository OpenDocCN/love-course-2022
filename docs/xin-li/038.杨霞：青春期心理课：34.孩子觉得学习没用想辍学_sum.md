# 【总结】038.杨霞：青春期心理课：34.孩子觉得学习没用想辍学

-   孩子觉得学习没用想辍学，家长感到困惑。一些学习成绩优秀的孩子突然想辍学，表现出学习倦怠和缺乏动力，原因是心理上出现了哭结，缺乏远大目标和价值感。
    1.  一些孩子觉得学习无用，想辍学，家长感到困惑。
    2.  学习成绩好的孩子也出现学习倦怠和缺乏动力的情况。
    3.  孩子缺乏远大目标和价值感，导致产生辍学想法。
-   孩子经历劳动后，对学习和劳动的看法发生改变
    1.  孩子发现没有知识的劳动更加痛苦
    2.  孩子重新选择学习，意识到学习的重要性
    3.  家长应该教育孩子承担责任，关注未来发展
-   责任教育对孩子的重要性
    1.  学习并非永远快乐轻松，会产生厌倦和疲惫
    2.  学习的意义在于能感受责任，看到未来理想
    3.  孩子通过建立责任和未来联系，坚持完成学业。
