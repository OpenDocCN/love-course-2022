# 【总结】038.杨霞：青春期心理课：37.孩子考完试抽搐怎么回事

-   孩子考完试出现抽搐，原因是神经紧张，缺乏运动和放松机制。
    1.  孩子爱学习但不喜欢运动，导致神经过度紧张。
    2.  家长忙于工作，建议增加户外活动和运动。
    3.  考试后紧张状态持续，缺乏放松机制导致抽搐发生。
-   孩子学习要劳力结合，保证全面发展，不仅要成绩好，还要爱运动爱劳动，保持健康。家长要合理安排学习时间，培养良好习惯，保证孩子运动量和休息时间。
