# 【总结】038.杨霞：青春期心理课：15.孩子情绪难以自控怎么办

-   孩子青春期情绪波动大，家长难以控制，需培养孩子自控能力
    1.  青春期孩子情绪波动大，容易冲动，家长难以应对
    2.  孩子要求尊重、隐私、不比较，家长需注意言行
    3.  青春期孩子思维不成熟，情绪激动，需培养协调性促进思维情绪沟通
-   孩子情绪和思维发展不平衡，家长难以控制自己情绪，但行为可控
    1.  孩子情绪和思维发展可能不平衡，导致行为冲动
    2.  人难以控制自己的思维和情绪，但行为可被控制
    3.  培养自我控制能力，避免冲动行为，维护良好人际关系
-   修养不佳时，接受情绪和想法，管理行动，顺其自然，为所当为，森田疗法八八字方针。接受情绪后，情绪会迅速消失，不要与情绪对立，不要为所欲为，否则成为动物或抗制情绪。
    1.  接受情绪和想法，管理行动，顺其自然，为所当为。
    2.  森田疗法八八字方针，接受情绪后迅速消失。
    3.  不要与情绪对立，不要为所欲为，否则成为动物或抗制情绪。
