# 06、魅力塑造：7.0网络课程第09节：框架

【一切都在你身上】，好，那这一节课我们讲一讲关于框架，什么是框架呢？，框架就是你的态度，你的行为准则，它还包含着你的三观，框架大部分的呈现是来自于，互动时话语背后潜在的含义，它折射出你内在的强与弱。

主导意或是被动，框架还包含了你对这个世界的设定，你对一段互动所下的含义，好的框架确实是能够吸引人，那软弱的框架只会让别人觉得你是个可怜虫，所以说无论是在把妹当中还是人际交往中，框架的保持永远是重中之重。

甚至有时候它比价值还要重要，那为什么说好的框架有时候比价值还要重要呢？，其实我们在这里举个例子，很简单，我们常说高价值产生吸引，其实这句话不是真正准确，真正准确的来讲是，框架控制高价值产生吸引。

其实在我周围有认识到一些朋友，还包括我们一些很有钱的学员，他们是有着非常非常高的价值，比如说我周围那些朋友，他出来吃饭的时候喜欢把扯钥匙摆在桌面上，但是他这样的行为却是在什么？，在炫耀。

在寻求别人的认同，他们需要用这些物质的东西来掩盖自己内心的空洞，以及不自信，其实一个真正高价值的人是不需要拿这些东西来证明自己的价值，所以这种炫耀，这种寻求别人的认同，这种行为会变成低价值。

就好像我们还有很多很有权的学员，他们以前有跟我发过他跟其他女生的聊天记录，就是他跟女生说我一个月花多少钱去包养你，我有多少多少钱，然后这个女生却拒绝了他，为什么？，因为他就是在炫耀。

他在炫耀自己家里有多少资产，他在炫耀自己的车，自己的房子，这是一种低价值，所以大家记住，框架控制高价值才会产生吸引，如果说你的高价值只是拿出来炫耀或者说是吹嘘夸扣，它只会变成一种低价值。

首先在我们PoA里面有一个很重要的观念就是，把妹是吸引而非追逐，你要明白的是你无法追赶到一个蝴蝶，你要做的或者说是你能做的，只有把自己变成一朵花然后吸引它，所以在你把一个妹子之前。

你一定要换掉一个意识框架，就是把妹是吸引而非追逐，从远古时代到现在的21世纪，女人的情感电路永远只会为那些强大的具有价值的男人所打开，一个具有自己框架的男人会显得更有价值，女人所喜欢男人的那些特质。

比如说自信、勇敢、敢于表达自己真实的见解，正是框架所折射出来的信息之一，举个例子，很多男生在碰到一些比较好看的姑娘的时候，往往就会这么说，哇，这么好看，有点不敢知识，这是一个极其缺乏自信的弱框架。

无论说是一句话还是一个念头，它都会折杀掉你和这个姑娘唯一的机会，而你应该做的是更具价值、更加准确的自我定位，你要明白的是内心的自信以及独定，跟你的财富、跟你的颜值是没有任何关系。

认可自己才是对你生命最好的尊重，举个例子，比如说女孩子问你，你喜欢我什么，大部分人由于有那个软弱的框架，有那个追逐的框架，普通男生都会说，因为你很漂亮，因为你会做饭，因为你很独立。

因为你跟其他女生不一样，因为等等等等，都是呈现出一个弱的框架，然后都是在追逐对方，那我们刚刚前面就说了，你应该拥有一个更加准确的自我定位，这个自我定位是更具价值的，所以在这个时候。

我们一定要运用到自己的框架，很简单，你还挺自信，再举个例子，很多男生大部分时候，由于呈现出讨好或是追逐的框架，在这些行为表达过后，往往会收到女生的一句话，这句话是什么呢，就是看你表现。

我相信有很多人都有收到过这句话，好 那女生说看你表现，咱们先不管她的前因是什么，如果说有这句话，那大家会怎么回呢，普通男人说，我一定会对你好的，我一定会照顾你，我一定会对你如何，置死不渝那些话。

那我们这一节讲的就是框架，所以我们当然不能像普通男生那样回，置死不渝等等等等，那我们这里可以怎么回呢，其实很简单，算了 我担心你会控制不住，因为对方说看你表现，其实这里用的是一个逆转框架。
