# 【总结】陈老师的《结婚课》：第二十六：约会时间和地点安排的学问

-   陈老师的《结婚课》第二十六讲强调约会时间和地点的重要性，提倡创造舒适感和熟悉感。选择适合交谈的地点，避免选择火锅店或嘈杂的环境，实地考察约会地点的安静程度和适宜性。
    1.  约会地点应选择适合交谈的环境，避免嘈杂和干扰。
    2.  第一次约会时，应多问问题，了解对方的工作、兴趣爱好和家庭情况。
    3.  通过问问题，提供话题和判断对方是否合适，帮助推进关系发展。
-   对话时间至关重要
    1.  对话时间需达到1.5-2小时
    2.  长时间对话带来舒适感和熟悉感
    3.  舒适感是建立关系的基础
-   对话时间是关系升级的必经过程
    1.  长时间对话有助于准备下一步
    2.  两个小时的对话是必须的
    3.  对话时间是关系发展的关键
