# 【总结】14.夫妻房事管理：学员学前准备

-   学习婚姻管理系统前的准备
    1.  不要强迫配偶一起学习
    2.  学习每个课程要三遍，做笔记
    3.  学习后要学以致用，慢慢谈问题
-   学习过程中的注意事项
    1.  不要急于应用，要熟练掌握
    2.  避免在冲突时谈论问题
    3.  提前准备好，放松状态下谈问题
-   学习后的分享和坚持
    1.  每天分享学习笔记的一部分
    2.  分享可以督促自己学习
    3.  坚持学习，不要因为配偶回家就停止学习
