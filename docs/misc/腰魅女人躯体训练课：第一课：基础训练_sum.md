# 【总结】腰魅女人躯体训练课：第一课：基础训练

-   第一课：基础训练
    1.  学习如何通过呼吸带动胸腔进行基础训练
    2.  呼吸的顺畅会影响身体的律动
    3.  通过温柔的呼吸练习，学会与身体建立连接，感受身体的柔软和舒适
-   第一个姿势的进阶训练
    1.  练习胸腔呼吸，进入更高级的训练阶段
    2.  自由发挥上半身的躯体，找到适合自己的训练方式
    3.  重点在于发现自己喜欢的方式，个性化训练，提高舒适度和效果
-   腰部和侧腰在运动时会相互影响

    2.  可以加入左右的波浪动作
