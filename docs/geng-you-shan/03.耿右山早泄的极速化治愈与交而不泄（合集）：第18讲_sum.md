# 【总结】03.耿右山早泄的极速化治愈与交而不泄（合集）：第18讲

-   耿右山探讨早泄、阳痿和性欲低下的治疗过程中，强调补足虚损是必要条件，但存在补不进去的情况。
    1.  补足虚损时遇到失热、失寒和瘀组等情况，难以补足。
    2.  病理性移经是补足虚损过程中的问题，会导致身体虚弱。
    3.  生殖器炎症和失热会引起病理性移经，需控制补虚过程，避免进一步虚弱。
